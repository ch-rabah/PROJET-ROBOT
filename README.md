# Projet Robot

## Description du Projet

Le projet consiste à réaliser un programme en groupe qui permettra de faire bouger un mini-robot. L'objectif principal est de s'organiser en groupe pour réussir à écrire un programme python sur lequel on implémentera sur carte raspberry pi.

### Objectifs principaux :
- Tracer un carré
- S'approcher le plus vite possible et le plus près d'un mur sans le toucher
- Suivre une balise

### Technologies utilisées :
- Un Raspberry Pi
- Une carte contrôleur (arduino)
- Deux moteurs encodeurs pour le contrôle des roues
- 3 senseurs : une caméra, un capteur de distance et un accéléromètre
